### Site web Dentics

il s'agit d'un site web statique pour un clique dentaire.

### le site web est créé avec :
1. HTML
2. CSS
3. JS

le depot contient :

1. Un fichier index.html
2. Un fichier READEME.md
3. Un dossier Image
4. Un dossier CSS
5. Un fichier signup.html
